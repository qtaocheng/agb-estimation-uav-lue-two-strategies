The contents shared in this repository are the core code for the two strategies and some of the test data in the UAV-LUE method for estimating wheat AGB. 
The meteorological parameters involved in the LUE model (PAR, PMIN, accumulated PMIN) and the LAI estimated by the BPNN-PROSAIL model have been pre-calculated and placed in the file "ACpar_data_agronomic_test.csv".
